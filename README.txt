A full-stack web application designed to modernize nuisance reporting for apartment
complexes. The system provides tenants with an intuitive interface for submitting reports,
sends automated email alerts to security, and offers management a centralized dashboard
for monitoring and exporting data. Built with Streamlit, Python, and deployed on Streamlit
Cloud, this project demonstrates expertise in web development, user experience design,
and business process automation.
